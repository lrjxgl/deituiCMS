
1.控制器
一般控制器与表名称一样 比如article表 文件article.ctrl.php
模板文件名称一般与action一样 除了default是index.html
default 首页  article/index.html
list  列表页	article/list.html
show  详情页面
my 我发布的
add 编辑页
save  保存数据
status 修改状态
delete  删除数据
pass 	审核通过 
forbid 	审核不通过
toggle	状态切换 比如收藏/取消收藏 

2.模型
	Dselect 选择已处理的列表项
	DselectRow 已处理的一行数据
	getListByIds();//根据id获取数据
	
3.变量
	变量采用驼峰 artList,在可识别基础上尽量简写
	规定列表主项为list变量
	其它主项为data变量
	所有api数据请求格式为:{
		"error":0,
		"message":"success",
		"data":{
			"list":[]
		}
	}
4.UI模板
	ui全部采用http://ui.deitui.com 
	所有页面必须先做成静态html，更改也从静态页面开始
	所有网页必须兼容小程序\weex\浏览器
